🧊 FridgeGeniusAI (SaveEat ✨)
AI·딥러닝 기반 스마트 냉장고 관리 & 음식 낭비 감소 플랫폼
1. 프로젝트 개요 (Project Overview)

FridgeGeniusAI는 냉장고 속 재료를 단순히 “기록”하는 애플리케이션이 아닙니다.
본 프로젝트는 딥러닝과 생성형 AI 기술을 활용해 사용자의 식재료 소비 의사결정을 지원하고, 음식 낭비를 실질적으로 줄이기 위한 지능형 서비스를 목표로 설계되었습니다.

현대 사회에서 음식물 쓰레기는 환경·경제·윤리적 측면에서 중요한 문제로 떠오르고 있습니다.
하지만 대부분의 냉장고 관리 앱은 다음과 같은 한계를 가지고 있습니다.

재료를 직접 하나하나 입력해야 하는 불편함

유통기한을 확인하는 데서 그치는 단순 관리 기능

“그래서 지금 뭘 먹어야 하는지”에 대한 판단 부재

FridgeGeniusAI는 이 문제를 AI 기반 자동 판단 구조로 해결하고자 합니다.

2. 문제 정의 (Problem Statement)
❗ 기존 냉장고 관리의 한계

정보는 있지만 판단은 사용자 몫

유통기한은 보이지만 어떤 재료가 더 급한지 판단하기 어려움

재료 입력 과정의 피로도

텍스트 입력은 귀찮고 누락되기 쉬움

AI 활용의 형식적 적용

단순 챗봇 수준의 레시피 추천에 머무름

실제 행동 변화로 이어지지 않음

“알지만 행동하지 않는” 문제 발생

5. 핵심 AI & 딥러닝 기능 상세
🧠 5.1 유통기한 기반 위험도 점수 모델

각 재료의 남은 유통기한을 기준으로 0~100점 위험도 점수 계산

단순 선형 계산이 아닌 구간별 가중치 모델 적용

상태	점수 범위	의미
안전	0~39	여유 있음
임박	40~69	주의 필요
위험	70~100	즉시 소비 권장

👉 이 점수는 모든 AI 판단의 기준 지표로 활용됩니다.

🤖 5.2 TOP 위험 재료 자동 추출

위험도 점수가 높은 재료를 자동 정렬

사용자가 따로 설정하지 않아도 “오늘 꼭 먹어야 할 재료 TOP 3” 제공

👉 사용자의 결정 피로도를 최소화하는 핵심 기능입니다.

🍳 5.3 AI 레시피 추천 (OpenAI GPT)

GPT API를 활용한 재료 기반 레시피 자동 생성

다음 조건을 반드시 만족하도록 프롬프트 설계

실제 조리가 가능한 레시피

가정용 재료/도구 기준

단계별 조리 과정 포함

AI 출력 예시

레시피 이름

예상 조리 시간

난이도

사용 재료

조리 단계

추가 팁 및 대체 재료

👉 단순 생성형 AI가 아닌 상황 인식형 AI 활용 사례입니다.

📸 5.4 사진으로 재료 인식 (딥러닝)

Google ML Kit Image Labeling 모델 사용

온디바이스 딥러닝 기반 이미지 분류

인식 결과 + 신뢰도(Confidence) 제공

⚠️ 딥러닝 오인식 문제를 고려해
→ 사용자가 직접 선택하여 냉장고에 추가하는 구조로 UX 설계

👉 AI 판단 + 사용자 통제를 결합한 현실적인 딥러닝 적용 사례입니다.

📊 5.5 인사이트 & 사용 패턴 분석

AI 기반 낭비 감소 지표 제공

예상 절약 비용 계산

위험 재료 관리 성과 표시

기능 사용 패턴 분석

👉 사용자가 “이 앱을 써서 어떤 변화가 있었는지” 확인 가능
